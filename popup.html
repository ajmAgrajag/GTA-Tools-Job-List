About GTA Tools - Job List

Welcome to GTA Tools - Job List, an extension designed to help you summarize and export job details from Rockstar Social Club.

How to Use

Follow these steps to use the extension:

  1. Open a compatible Rockstar Social Club page in your browser. Supported pages include:
    • Job pages: Example: .../job/gtav/...
    • Playlist pages: Example: .../games/gtav/ps5/playlists/playlist/...
  2. Make sure the page is fully loaded before using the extension. (For playlists, the extension will open multiple tabs to load jobs. You'll need to wait for these to fully load before re-clicking the extension icon)
  3. Click the extension icon to view job details. Use the Generation Tab to export data in BBCode, Text, CSV, or "YouTube" formats.
  4. Customize the exported fields in the Settings Tab.
